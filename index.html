Featured project

Customer-Support Triage Agent

An open-source multi-step agentic workflow: it classifies a ticket, retrieves grounded context from a Chroma vector store (RAG), drafts a reply, and evaluates its own confidence before sending — escalating to a human when it isn't sure. Vague tickets route to a ReAct sub-agent that gathers context with tools first. Every node emits a LangSmith trace.
